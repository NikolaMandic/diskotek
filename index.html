diskotek

early development version of ui for gdb debugger for arm on linux !!! some code is not as it should be right now because I wanted to implement some features faster. unstable now but in active development

docs

http://nikolamandic.github.io/diskotek/

ubuntu setup

for tools needed check out /vdir/bootstrap.sh
that is a script for bootstraping fresh installation of ubuntu precise so it can run this in vm

some of the steps needed
sudo apt-get install gcc-arm-linux-gnueabi
sudo apt-get install qemu-user-static
sudo apt-get install gdb-multiarch
export QEMULDPREFIX=/usr/arm-linux/gnueabi

latest nodejs

sudo nodejs ws.js

open localhost:3000/index.html#/ in browser and click debug

vm debugging

sudo apt-get install vagrant

cd vdir
vagrant up
go to localhost:8080/index.html#/
now app is being debugged inside a virtual machine
